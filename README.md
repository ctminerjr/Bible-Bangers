# Bible Banger (PWA)
A tiny offline-capable web app you can install on your iPhone Home Screen.

## Deploy to GitHub Pages
1. Create a new repo and upload all files in this folder (including `icons/`).
2. In the repo: **Settings → Pages → Build and deployment**
   - Source: **Deploy from a branch**
   - Branch: **main** (or `master`), folder: **/ (root)**
3. Open the Pages URL it shows you.

## Add to Home Screen (iPhone)
1. Open the site in **Safari** on your iPhone.
2. Share → **Add to Home Screen** → **Add**.
3. Launch it from your Home Screen. Works offline after first load.

Data is stored locally (in `localStorage`) on the device.
